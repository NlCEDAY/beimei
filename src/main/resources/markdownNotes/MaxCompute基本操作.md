## MaxCompute基本操作：

- 登录客户端shell界面

```shell
odpscmd
```

- 查看命令

```shell
help
```

### 常见命令：

- 切换项目

```sql
use xxxx
```

- 查看可用表

```sql
list tables
```

- 查看可用资源

```sql
list resources
```

- 查看可用函数

```sql
list functions
```

- 查看表结构

```sql
desc xxx
```

- 创建表

```sql
create table mydemo(id int,name string)
```
